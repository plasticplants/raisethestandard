Intro

This website is a tool to help people design a new flag for the state of Mississippi. You can use the current pieces and rearrange them into something better, or you can from scratch and create something completely new.

If you are looking for a place to start, check out the resources for info on Mississippi's history, good flag design principles, and great inspiration.
